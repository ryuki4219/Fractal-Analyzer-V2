## ⚡ 超高速セットアップ (3ステップ)

### Step 1: インストール (2分)

```powershell
# 1. リポジトリをクローン
git clone https://github.com/ryuki4219/Fractal-Analyzer-V2.git
cd Fractal-Analyzer-V2

# 2. 仮想環境を作成・有効化
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 3. 依存パッケージをインストール
pip install -r requirements.txt
```

### Step 2: 起動 (30秒)

```powershell
# アプリを起動
streamlit run fractal_app.py
```

または、`起動.bat`をダブルクリック

### Step 3: 使い始める (2分)

ブラウザが自動的に開きます → すぐに使えます! 🎉

## 💡 よくある最初の質問

### Q: モデルファイルがないと言われる
**A**: 先に学習モードでモデルを作成してください

### Q: 画像ペアが検出されない
**A**: ファイル名を確認:
- 正: `IMG_5023.jpg` ⇔ `IMG_5023_low1.jpg`
- 誤: `IMG_5023.jpg` ⇔ `IMG_5024_low1.jpg` (番号違い)

### Q: 精度が低い (85%以下)
**A**: 以下を確認:
1. データ拡張は15種類以上か
2. 品質レベルは適切か (推奨: low1)
3. 画像数は十分か (100組以上推奨)
